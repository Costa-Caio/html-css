@font-face permite que você use qualquer fonte (como uma fonte baixada do Google Fonts ou da internet), mesmo que ela não esteja instalada no computador de quem acessa o site.
@font-face { font-family: 'MinhaFonte'; src: url('fonts/MinhaFonte.ttf') format('truetype'); } body { font-family: 'MinhaFonte', sans-serif; }
